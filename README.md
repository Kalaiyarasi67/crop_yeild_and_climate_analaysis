🌾 Crop Yield and Climate Analysis Dashboard
📌 Project Overview
The Crop Yield and Climate Analysis Dashboard is an interactive Excel dashboard designed to analyze agricultural production and climate conditions across multiple locations and seasons.
This project helps visualize:
Crop yield trends
Seasonal performance
Climate impact on agriculture
Revenue analysis
Location-wise comparisons
The dashboard is built using Microsoft Excel with Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques.
🎯 Objectives
Analyze crop yield patterns over different years
Understand the relationship between climate and agriculture
Compare seasonal agricultural performance
Create an interactive and user-friendly dashboard
Improve decision-making using data visualization
